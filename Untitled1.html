Responsive Layout 2

Your file for this exercise should be saved in the public folder and should be accessed using this link: "/css-exercises/responsive-layout-2.html". The content of the HTML file is given below. With the given HTML content, design the web page using an internal style sheet in such a way it will look exactly like the screenshot below.
Note: You cannot modify the content, only the style sheet AND the classes!
Aside from the design visible in the screenshot, here are some additional details for the design of the web page:
  • All of the elements should have the box-sizing property set to border-box.
  • Make use of meta tag to specify the author and description of your exercise.
  • Make use of meta tag to set the width of the page to follow the screen-width of the device and set the initial zoom level when the page is first loaded by the browser.
  • The CSS rule for the column sizes are already included in the HTML page, you just need to set the right classes for each div tag (column) to fix the display of the page when viewed on desktop, tablet and mobile phone (See the 3 screenshots).
